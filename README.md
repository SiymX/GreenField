# Green Field


A lightweight 3D scene built with Three.js that renders a scenic field with a house in the middle.



---

Test the live demo here: https://siymx.github.io/GreenField/
---
## Features
- Procedural ground and grass
- Adjustable wind and density
- Tree placement controls
- Weather modes: Clear, Misty, Stormy (with rain in Stormy)
- Time of day control (0â€“24)
---
## Controls
- Orbit: left-drag
- Pan: right-drag
- Zoom: mouse wheel
